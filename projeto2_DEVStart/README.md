# Projeto de certificação 2 DEVStart

## Tecnologia utilizada:
![Javascript](./assets/js.png)
### O projeto recebe, formata e calcula diversos dados de atletas.

## Funcionalidades do projeto:
- [X]  Recebe informações de atletas através de programação orientada a objetos.
- [X] Classifica a categoria do atleta pela faixa etária.
- [X] Cálculo do IMC.
- [X] Cálculo da nota média válida, descartando a nota maior e menor.
- [X] Entrega um relatório completo do atleta.


## O código pode ser executado em:
Navegadores web modernos: Chrome, Microsoft Edge, Safari, Brave, etc. Ou Node.js.

## Como executá-lo?

### Execução em navegadores web:
- Salve o código em um arquivo com o nome dados-atletas  e cole todo o código do projeto.
- Crie um arquivo .html
- Abra esse arquivo html em um navegador web moderno.
----------------------------------------------------------
### Execução no Node.js:
- Salve o código em um arquivo com o nome dados-atletas.
- Execute no terminal.
